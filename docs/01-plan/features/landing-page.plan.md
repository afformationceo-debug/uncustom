## 1. 개요 (Overview)

### 1.1 목표
Uncustom 인플루언서 마케팅 자동화 플랫폼의 **서비스 소개 랜딩페이지**를 개발한다. AI 기반 키워드 추출부터 캠페인 관리까지 전 기능을 **고객 관점의 설득적 스토리텔링**으로 전달하며, **3D 느낌의 프리미엄 디자인과 고급 애니메이션**으로 벤치마크 대비 압도적인 비주얼 임팩트를 구현한다.

### 1.2 핵심 요구사항
- 모든 기능을 고객 관점에서 **페인포인트 → 솔루션** 플로우로 설득
- 3D 글래스모피즘, 파티클, 그라데이션 등 **프리미엄 비주얼 효과**
- Framer Motion 기반 **스크롤 트리거 애니메이션**
- "무료 체험하기" CTA → 실제 `/login` 페이지 연결
- 소개서 다운로드는 미구현 (추후 추가)
- **모바일 반응형** 완벽 지원
- **다크 테마** 기반 (벤치마크 spray.io 참고)

### 1.3 벤치마크 대비 차별화 전략

| 항목 | spray.io | altari.ai | **Uncustom (목표)** |
|------|----------|-----------|---------------------|
| 테마 | 다크 (gray-900) | 라이트 (퍼플 그라데이션) | **다크 + 퍼플 네온 글로우** |
| 애니메이션 | 기본 전환 | Framer Motion 기본 | **3D 트랜스폼 + 파티클 + 패럴랙스** |
| 히어로 | 텍스트 + CTA 2개 | 텍스트 + 제품 스크린샷 | **3D 플로팅 UI 모킹 + 파티클 배경 + 타이핑 애니메이션** |
| 기능 소개 | 카드 5개 나열 | 카드 3열 그리드 | **인터랙티브 스텝 애니메이션 (스크롤 연동)** |
| 소셜 프루프 | 인플루언서 카드 3개 | 고객사 로고 마퀴 + 후기 | **라이브 데모 느낌 대시보드 + 수치 카운트업** |
| CTA | 데모 체험 + 소개서 | Book a Call | **네온 글로우 CTA + 호버 3D 틸트** |
| 전체 느낌 | 깔끔하지만 평범 | SaaS 전형적 | **사이버펑크 프리미엄 + 압도적 비주얼** |

## 2. 벤치마크 분석 요약

### 2.1 spray.io/ko (스프레이)

**강점:**
- PAS 구조 (Problem-Agitate-Solve) 카피라이팅
- "시간 효율 30배" 같은 구체적 수치 제시
- 에이전시 서비스 섹션으로 부가가치 제안
- 블로그 연동으로 SEO + 신뢰감
- 인플루언서 카드 (프로필 + 팔로워 + 콘텐츠 이미지) 소셜 프루프

**약점:**
- 애니메이션이 거의 없음 (정적)
- 영상 데모 2개뿐, 제품 스크린샷 부재
- 기능 카드가 단순 텍스트 나열
- 히어로 섹션 비주얼 임팩트 부족

**참고할 점:**
- 네비게이션: 로그인 + CTA 2단계 구조
- PAS 카피라이팅 플로우
- 인플루언서 카드 디자인 (프로필사진 + 팔로워수 + 시간)
- 푸터: 회사 정보 + 뉴스레터 CTA

### 2.2 altari.ai (알타리)

**강점:**
- Pill형 플로팅 네비게이션 (세련됨)
- 퍼플-블루 색상 체계 (SaaS 트렌드)
- 고객사 로고 마퀴 배너 (신뢰감)
- 비교 테이블 (vs 경쟁사) 섹션
- 4단계 프로세스 (01→02→03→04) 시각화
- FAQ 아코디언
- 팀 소개 섹션 (인간적 신뢰)
- 성과 수치 4개 (카운트업 효과)

**약점:**
- 영어 전용 (한국 시장 미대응)
- 제품 실제 스크린샷 제한적
- 기능별 세부 설명 부족

**참고할 점:**
- 수치 기반 성과 섹션 ($1.2M+, 60+ Hours, 3.5X, 30 Days)
- 인터랙티브 원형 다이어그램 (통합/연동 시각화)
- 플로팅 네비 디자인
- Testimonials + 고객사 로고
- How it works 4단계 프로세스

## 3. 페이지 구조 및 섹션 설계

### URL 구조
```
/ (메인 대시보드 - 기존 앱)
/landing (서비스 소개 랜딩페이지 - 신규)
```
→ 랜딩페이지는 `/landing` 경로에 독립 배치, 인증 불필요 (public route)
→ middleware.ts에 `/landing` public path 추가 필요

### 3.1 전체 섹션 순서 (12개 섹션)

```
[1] 네비게이션 (플로팅 글래스 네비)
[2] 히어로 섹션 (3D 플로팅 UI + 파티클 + 타이핑)
[3] 소셜 프루프 바 (로고 마퀴 + 수치 카운트업)
[4] 페인포인트 섹션 (문제 제기 → 솔루션 전환)
[5] 핵심 프로세스 섹션 (글로벌추출→마스터→캠페인 3단계)
[6] 기능 쇼케이스 - AI 추출 (키워드/태그 기반 5개 플랫폼)
[7] 기능 쇼케이스 - 캠페인 자동화 (제안서/템플릿/이메일/성과)
[8] 라이브 데모 섹션 (대시보드 목업 인터랙션)
[9] 비교 테이블 (Uncustom vs 수동 작업 vs 경쟁 솔루션)
[10] 성과 수치 섹션 (카운트업 애니메이션)
[11] FAQ 섹션 (아코디언)
[12] 최종 CTA + 푸터
```

---

### 3.2 섹션별 상세 설계

#### [1] 플로팅 글래스 네비게이션

**디자인:**
- `position: fixed`, 상단 중앙 플로팅 (pill 형태)
- 글래스모피즘: `backdrop-blur-xl`, `bg-white/5`, `border border-white/10`
- 스크롤 시 배경 불투명도 증가 애니메이션
- 로고 좌측 + 메뉴 중앙 + CTA 우측

**메뉴 항목:**
- 기능 소개 | 프로세스 | 성과 | FAQ
- CTA: **"무료 체험하기"** → `/login` 링크

**인터랙션:**
- 스크롤 다운 시 네비 축소 + 배경 진하게
- 현재 섹션 하이라이트 (Intersection Observer)
- 모바일: 햄버거 메뉴 → 풀스크린 오버레이

---

#### [2] 히어로 섹션

**레이아웃:** 풀스크린 (100vh), 중앙 정렬

**배경:**
- 다크 배경 (`#0a0a0f`) + 퍼플/블루 방사형 그라데이션 오버레이
- CSS 파티클 효과 (떠다니는 점/선, canvas 또는 CSS)
- 미세한 그리드 패턴 오버레이 (`bg-[url('/grid.svg')]`)

**메인 카피:**
```
[서브 헤드] AI가 찾고, 자동으로 섭외하는
[메인 헤드] 인플루언서 마케팅의
            모든 것을 자동화하세요
[설명] 키워드 하나로 Instagram, TikTok, YouTube, Twitter까지
      5개 플랫폼 인플루언서를 AI가 즉시 발굴하고,
      제안서 발송부터 성과 추적까지 한 곳에서 관리합니다.
```

**CTA 버튼:**
- Primary: **"무료 체험하기"** → `/login`
  - 네온 퍼플 글로우 + 호버 시 3D 틸트 + 쉬머 효과
- Secondary: **"기능 자세히 보기"** → 스크롤 to 기능 섹션
  - 글래스 스타일 아웃라인 버튼

**3D 플로팅 UI 목업:**
- 히어로 우측(또는 하단)에 제품 대시보드 스크린샷
- CSS `perspective` + `rotateY(-5deg) rotateX(5deg)` 3D 기울임
- 플로팅 카드들 (인플루언서 프로필, 이메일 템플릿, 성과 차트)이
  메인 대시보드 주위에 떠다니는 효과
- 각 카드 `animation: float 6s ease-in-out infinite` (시간차)

**타이핑 애니메이션:**
- 메인 헤드 일부가 타이핑되는 효과
- 또는 "인플루언서 마케팅" 부분이 "시딩 마케팅" → "브랜드 협업" → "인플루언서 마케팅"으로 교체

**스크롤 인디케이터:**
- 하단 중앙에 마우스/화살표 아이콘 바운스 애니메이션

#### [3] 소셜 프루프 바

**디자인:**
- 다크 배경에 미세한 분리선
- 좌측: "1,000+ 브랜드가 신뢰하는" 텍스트
- 우측: 로고 마퀴 (자동 스크롤)
  - 가상 브랜드 로고 또는 "Coming Soon" 플레이스홀더

**수치 3개 (카운트업):**
```
50,000+         5개 플랫폼       10배
추출된 인플루언서  동시 지원       시간 절약
```
- Intersection Observer로 뷰포트 진입 시 숫자 카운트업 애니메이션

#### [4] 페인포인트 섹션 (PAS 구조)

**디자인:** 다크 배경, 텍스트 중심, 강조색 하이라이트

**카피라이팅:**
```
[서브] 이런 고민, 하고 계시죠?

[카드 1] 😩 인플루언서 찾느라 하루종일 인스타 서핑
         "키워드 검색 → 프로필 확인 → 이메일 찾기... 한 명당 15분?"

[카드 2] 📧 DM/이메일 일일이 보내고 답장 기다리기
         "100명에게 같은 내용 복붙, 읽었는지도 모르는 이메일"

[카드 3] 📊 캠페인 성과? 엑셀 수동 집계
         "게시물 올렸는지 확인하고, 조회수 하나하나 기록하고..."

[전환] ↓

[솔루션 헤드]
         Uncustom이 이 모든 과정을
         AI로 자동화합니다

[설명]   키워드 입력 한 번이면 끝.
         AI가 인플루언서를 찾고, 제안서를 보내고, 성과를 추적합니다.
```

**애니메이션:**
- 페인포인트 카드: 스크롤 시 좌→우 슬라이드인 (stagger)
- 전환 시 배경이 다크 레드 틴트 → 퍼플 글로우로 변화
- 솔루션 텍스트: 타이핑 또는 페이드업 효과

#### [5] 핵심 프로세스 섹션 (3단계 파이프라인)

**디자인:** 가로 3단계 프로세스 시각화 (3D 카드)

```
[STEP 1]                    [STEP 2]                    [STEP 3]
글로벌 추출                  마스터데이터                  캠페인 배정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
키워드/태그 입력 하나로       추출된 인플루언서를            원하는 인플루언서를 선택,
Instagram, TikTok,          한 곳에서 필터링하고           캠페인에 배정하면
YouTube, Twitter, 전체를    팔로워수, 국가, 카테고리로     제안서 발송부터 성과 추적까지
AI가 동시에 스캔합니다       정밀하게 분류합니다            자동으로 진행됩니다

       ──────────→                 ──────────→
```

**3D 카드 디자인:**
- 각 스텝: 글래스모피즘 카드 + 상단 아이콘 (3D 느낌)
- 호버 시 카드가 약간 떠오르는 효과 (`translateY(-8px)` + `shadow-2xl`)
- 카드 간 연결선: 애니메이션 대시 라인 (SVG path)
- 스크롤 시 좌→우 순차 등장

**아이콘:**
- Step 1: 검색 + AI 아이콘 (글로우 효과)
- Step 2: 데이터베이스 + 필터 아이콘
- Step 3: 로켓 + 체크 아이콘

#### [6] 기능 쇼케이스 - AI 추출 엔진

**헤딩:**
```
AI 추출 엔진
키워드 하나로 5개 플랫폼을 동시에 스캔합니다
```

**좌측: 텍스트 + 기능 리스트**
```
✅ 키워드 기반 해시태그 검색 (Instagram #뷰티 #여행 등)
✅ 태그 계정 기반 경쟁사 인플루언서 발굴
✅ AI 자동 프로필 보강 (팔로워, 이메일, 바이오)
✅ 바이오 링크에서 이메일 자동 추출
✅ 국가/언어/카테고리 필터링
```

**우측: 인터랙티브 데모 목업**
- 플랫폼 아이콘 5개가 중앙 Uncustom 로고를 둘러싼 **원형 다이어그램**
  (altari.ai의 인터랙티브 다이어그램 참고하되 3D 글로우 버전)
- Instagram / TikTok / YouTube / Twitter / All 아이콘이
  네온 라인으로 중앙에 연결
- 호버 시 해당 플랫폼 하이라이트 + 설명 팝업

**애니메이션:**
- 스크롤 진입 시 중앙 로고 먼저 등장 → 플랫폼 아이콘 방사형 확산
- 각 연결선이 네온 펄스로 활성화
- 추출 결과 카드가 하단에서 슬라이드업

#### [7] 기능 쇼케이스 - 캠페인 자동화

**헤딩:**
```
캠페인 자동화
추출부터 성과까지, 원스톱으로 관리하세요
```

**4개 기능 카드 (2x2 그리드):**

| 카드 | 제목 | 설명 | 비주얼 |
|------|------|------|--------|
| 1 | 제안서 링크 빌더 | 브랜드 맞춤 제안서를 폼으로 작성, 공개 URL 자동 생성. 인플루언서가 링크로 미션/리워드 확인 후 바로 참여 신청 | 제안서 랜딩페이지 미니 목업 |
| 2 | DM/이메일 자동 발송 | 개인화 태그(이름, 팔로워수 등) + 제안서 링크를 자동 삽입하여 대량 발송. 열람/클릭 실시간 추적 | 이메일 편집기 + iPhone DM 미리보기 |
| 3 | 인박스 & 회신 관리 | 인플루언서 회신을 스레드별로 관리. 답장 상태 자동 업데이트 | 인박스 UI 미니 목업 |

## 5. 기술 구현 계획

### 5.1 기술 스택

| 레이어 | 기술 | 용도 |
|--------|------|------|
| 프레임워크 | Next.js App Router | SSR + 정적 생성 |
| 애니메이션 | **Framer Motion** | 스크롤 트리거, 페이지 전환, 레이아웃 |
| 3D 효과 | **CSS Transform** + Framer Motion | perspective, rotateX/Y, translateZ |
| 파티클 | **CSS 기반** (@keyframes) | 배경 파티클 (라이브러리 미사용) |
| 카운트업 | **Custom Hook** | Intersection Observer + requestAnimationFrame |
| 아이콘 | **Lucide React** | 기존 프로젝트 통일 |
| 폰트 | **Pretendard Variable** | 한국어 최적화 |

### 5.3 성능 최적화

| 항목 | 전략 |
|------|------|
| **이미지** | Next/Image + WebP/AVIF, lazy loading, blur placeholder |
| **폰트** | Pretendard 서브셋 + `font-display: swap` + next/font |
| **애니메이션** | `will-change: transform`, GPU 가속, `useReducedMotion` 존중 |
| **코드 분할** | 각 섹션 동적 import (`next/dynamic`) |
| **SSR** | 서버 컴포넌트 기본, 클라이언트는 인터랙션 필요 시에만 |
| **파티클** | CSS 전용 (JS 라이브러리 미사용), 모바일에서 감소 |
| **LCP** | 히어로 이미지 priority preload |
| **CLS** | 모든 섹션 min-height 사전 지정 |

### 5.4 반응형 전략

| 브레이크포인트 | 대응 |
|---------------|------|
| ≥1280px | 데스크탑 풀 레이아웃, 2열 그리드, 3D 효과 풀 |
| 768-1279px | 태블릿, 1-2열 적응, 카드 축소 |
| <768px | 모바일, 1열 스택, 3D 효과 축소, 파티클 감소 |

### 5.5 접근성 (A11y)

- `prefers-reduced-motion: reduce` 시 애니메이션 최소화
- 모든 이미지 `alt` 텍스트
- CTA 버튼 `aria-label`
- 키보드 네비게이션 지원
- 충분한 색상 대비 (WCAG AA)
